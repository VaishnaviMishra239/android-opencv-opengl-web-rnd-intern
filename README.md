# EdgeDetectorRND - Full-featured
Android + OpenCV (C++) + OpenGL ES 2.0 + JNI + TypeScript viewer.

## What is included
- Java Android app using Camera2 API
- JNI bridge to native C++ using OpenCV (Canny edge detection)
- OpenGL ES renderer to display processed frames
- TypeScript web viewer (static sample)
- CMake configuration and Gradle hints

## Quick setup
1. Install Android Studio with NDK and CMake.
2. Download OpenCV Android SDK and copy `sdk/native/jni` into `app/src/main/cpp/opencv` or place `.so` in `app/src/main/jniLibs/<abi>/`.
3. Open project in Android Studio, sync Gradle, build and run on a physical device.
